# Noctua Docs Template

## Usage

### 1. Customize

To edit configuration values, customize the _config.yml

To add pages, write them into the pages

You define urls based on the `permalink` attribute in your pages,
and then add them to the navigation by adding to the content of _data/toc.myl

The top navigation is controlled by _data/navigation.yml


### 2. Serve

Depending on how you installed jekyll:

```bash
jekyll serve
# or
bundle exec jekyll serve
```
